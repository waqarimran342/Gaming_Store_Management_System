# 🎮 Gaming Store Management System (C++)

## 📌 Project Overview
This is a console-based Gaming Store Management System developed in C++.  
The system simulates a real-world gaming store where users can browse games, purchase them, and receive discounts through an interactive game.

---

## 🚀 Features

- 🔐 Admin Login System
- 🤖 Human Verification (Anti-bot check)
- 📂 12 Game Categories
- 🔥 Trending Games Section
- 🛒 Game Purchase System
- 🎯 Discount Guessing Game
- 👤 Customer Information Collection
- 🧾 Bill Generation System

---

## 🧠 Concepts Used

- Arrays (1D & 2D)
- Structures
- Functions
- Loops & Conditions
- Random Number Generation
- File-like Data Handling (static)

---

## 🛠️ Technologies

- Language: **C++**
- Compiler: Any (CodeBlocks, Dev C++, VS Code, etc.)
